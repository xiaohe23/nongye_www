# 文档说明

## 介绍
enseCAP云平台主要功能是管理物联网传感器设备和存储采集到的数据，用户可以申请账号，并将设备统一绑定在该账号下，SenseCAP提供基于Web的管理平台和API接口，Web云平台包含仪表盘、设备管理、数据管理、安全管理等功能模块，API主要是用于用户做二次开发时调用。
